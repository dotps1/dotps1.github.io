Functions

A collection of functions I have personally developed to overcome situations I have encountered.
  ConvertTo-ShortPath - Converts a file or folder path to the MS DOS 8.3 short path equivilent.
  Export-AclToCsv - Exports the Acl Object for files and or folders to an easy readable csv, helps to 'show' what the permission hiearchy truly is.
  Get-ADObjectWithLinkedGPO - Gets all objects through out a domain that have a particular Group Policy Object applied to it. This can be filtered to return Computers, Group, or Users Objects only.
  Get-LastLoggedOnUser - Returns a PSCustomObject of the last logged on user or currently logged on user from an accessable workstation.
  Get-SqlEdition - Returns all instance and edition types of SQL from a machine.
  Get-UnEncryptedWorkstationsFromCMDB - Returns a collection of CIs that are not encrypted from the Microsoft System Center Configuration Manager database.
  Invoke-SophosReinit - Used to remotly execute the Sophos Reinit, namely to point clients to a Managment Server.
  Test-Credential - Returns a bool repsentation of a PSCredential objects validity.
  Update-NetworkPrinterServerConnection - Updates the Print Server connection info of network printer mappings. Useful when replaceing print servers.
  Get-ADComputerSiteName - Gets the AD Site name associated to a computer.
  New-Username - Creates a new Active Directory based user name.
  Get-MsiProductCode - Opens a Windows Installer Database (.msi) and querys for the product code.
  Get-DnsConfiguration - Retrives primary, secondary, tertiery DNS Servers from on online system using Windows Management Instrimentation.
